Handles rendering the map grid tiles and placement previews.

Loops through the 2D map array to draw static environmental blocks adjusted for camera scroll. If a user is currently plotting a two-stage path object (like a walker or moving platform), it overlays a temporary ghost tile at the designated start coordinates.

Loads a saved file composition back into memory.

Attempts to locate and parse a level file matching the targeted global index. If found, it safely resets any active geometry arrays, updating the editor grid state back to the historical save point.
