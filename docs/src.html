src.main module¶

src.main.main() → None[source]¶

Handles loading different parts of the game.

Initializes the Pygame modules, sets up the display window, and manages the core game loop. It acts as a state controller, continuously switching between menus, the level editor, and active gameplay based on the value of the global game state.
